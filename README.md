# SE Laboratory Midterm Exam – Calculator Module

## Description
A simple Python calculator module developed for the Software Engineering Laboratory Midterm.
The project demonstrates clean code, automated testing, and CI/CD using GitHub Actions.

## Features
- Addition and safe division
- Input validation and error handling
- Automated unit tests using pytest
- Continuous Integration with GitHub Actions

## Project Structure

src/ # Application logic
tests/ # Automated tests
.github/ # CI workflow

## How to Run Tests
bash
pip install -r requirements.txt
pytest
